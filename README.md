# Repositório de Estudos em Front-End

Este repositório reúne minhas atividades, exercícios e projetos desenvolvidos durante meus estudos em **Front-End**.

Aqui você encontrará desde práticas básicas até projetos mais estruturados utilizando tecnologias modernas do desenvolvimento web.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* React
* Vite
* Git & GitHub

---

## Estrutura do repositório

O repositório está organizado em pastas conforme o tipo de atividade:

```bash
aulas/              # Conteúdos e exemplos das aulas
lab/                # Testes e experimentos
three-components/   # Projeto React com Breadcrumbs
```

---

## Projetos em destaque

### Breadcrumbs em React (`three-components`)

Projeto que demonstra diferentes formas de estilização em React:

* CSS tradicional
* CSS Modules
* Styled Components

#### Funcionalidades:

* Breadcrumb interativo
* Navegação com estado (`useState`)
* Botões de avançar e voltar

---

## Como executar os projetos

Cada projeto possui suas próprias dependências. Exemplo com Vite:

```bash
cd nome-do-projeto
npm install
npm run dev
```
